# Simple FPS Macro

> *Use with caution: I am not responsible for any in-game bans or violations*

A lightweight C++ console-based recoil macro for FPS games. It comes with a few options of low, medium, and high recoil control. 

<img width="1100" height="573" alt="program" src="https://github.com/user-attachments/assets/a8f53bcb-bd7a-4935-9ecb-d6f8d33874df" />

## Usage
It can be used in almost any game as a simple yet effective recoil macro that is fully customisable to your need! 


## Installation

1. Install the ZIP file provided on the repository
2. Compile main.cpp using a C++ compiler of your choice

```bash
g++ main.cpp -o name
```

3. Execute the .exe with the name you compiled it as from the cpp file. (It is recommended to run as Administrator if possible)
